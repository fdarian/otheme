# Generated target pages — do not hand-edit

Every `pages/targets/*.mdx` file is GENERATED by `apps/docs/scripts/generate-target-pages.ts` and overwritten on `bun run generate` (which runs automatically via the `predev`/`prebuild` hooks in `apps/docs/package.json`).

To change a target page, edit the generator, not the `.mdx`:
- Per-target prose lives in that target's `TargetPageConfig` entry (`intro`, `compatibility`, `setupSection`, `midSections`).
- The Files/Commands tables come from the adapter's `plan()` in `packages/core/src/adapters/<target>-adapter.ts`.

After editing, run `bun run generate` and commit the generator change together with the regenerated `.mdx`.
