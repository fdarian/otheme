# otheme

Monorepo for the otheme CLI and theme adapter packages.

## Workspaces
- `packages/config` - shared TypeScript configs.
- `packages/core` - theme spec, registry, alias store, and target adapters.
- `apps/cli` - user-facing `otheme` command.

## Commands
- `bun install`
- `bun run check`
- `bun run format`
